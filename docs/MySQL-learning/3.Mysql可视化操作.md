# 安装phpMyAdmin

![image](https://cdn-mineru.openxlab.org.cn/result/2026-03-26/6b805dfb-b345-42fa-9c1c-a5a5f21bfe1a/3be905d000987dc98959183270e8149a64e89d55fc21e68d0e4c41f82ac5d6d2.jpg)


phpMyAdmin 是一个网络接口，通过它可以管理你的 MySQL 数据库。

apt-get install phpmyadmin 

你会看到以下问题：

Web server to reconfigure automatically: 

选择 apache2，按确定。

# Configure database for phpmyadmin with dbconfig-common?

![image](https://cdn-mineru.openxlab.org.cn/result/2026-03-26/6b805dfb-b345-42fa-9c1c-a5a5f21bfe1a/f9c4eaf40cae3fc8005a68ec7ee16f6edd65da547387b8e5632505acd1b80b51.jpg)


选择 No，按确定。

属性配置

将/usr/share 目录下的 phpmyadmin 文件夹链接到/var/www/html/

In -s /usr/share/phpmyadmin/ /var/www/html/phpmyadmin 

打开phpMyAdmin管理页面

http://localhost/phpmyadmin/ 

注意：localhost 即你的 ubuntu IP 地址。

就可以访问phpmyadmin页面了，如下图示：

phpMyAdmin

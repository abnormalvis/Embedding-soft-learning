# 结构体对象

MYSQL结构

结构代表一个数据库连接句柄，包含有关服务器的连接状态的信息，几乎所有函数都是用到它。

```txt
MYSQL mysql; 
```

MYSQL_RES 结构

代表返回行的查询结果（SELECT、SHOW、DESCRIBE等），从数据库读取数据。

```txt
MYSQL_RES *res; 
```

MYSQL_ROW 结构

通过调用获得行 mysql_fetch_row()。它目前实现为计数字节字符串数组。

```txt
MYSQL_ROW row; 
```

# 函数接口

# 1）连接/断开服务器

mysql_init(MYSQL\*); //分配或初始化MYSQL适合的对象

mysql_real_connect //尝试建立与运行的MySQL数据库引擎的连接host。

```c
MYSQL *STDCALL mysql_real_connect(MYSQL *mysql, const char *host, const char *user, const char *passwd, const char *db, unsigned int port, const char *unix_SOCKET, unsigned long clientflag); 
```

```txt
mysql_close(MYSQL\*); //关闭数据库连接  
mysql_select_db(&mysql,数据库名) //选定当前数据库
```


示例：连接 mysql_server


```c
MYSQL mysql;   
mysql_init(&mysql);   
//mysql_real_connect(MYSQL \*,"192.168.1.44","root","123456",""",3306,NULL,0)   
if(!mysql_real_connect(&mysql,0,"root","123456",NULL,0,NULL,0))   
{ printf("connet failed\n"); exit(1);   
}   
printf("connect success!\n\n"); 
```


//选择数据库



# 2）执行SQL语句

mysql_real_query(); 

//执行指向的SQL语句，字符串length字节长。

mysql_query(MYSQL *, char *ptr); 

//执行以 null 结尾的字符串指向的 SQL 语句，成功返回 0，失败返回 1

# 4）结果集中列操作（字段操作）

unsigned short mysql_num_fields(MYSQL_RES *); 

//从结果集中统计下"表的字段"个数（列数）

mysql_fetch_field(MYSQL_RES *); 

//从结果集中获取表的字段结构体，按左到右读取出来

示例：

unsigned short numFields; 

char column[30][40]; 

```c
numFields = mysql_num_fields(res);  
printf("%d\n", numFields);  
//保存字段名  
for(int i = 0; i < numFields; ++i)  
{  
    strcpy(column[i], mysql_fetch_field(res) ->name);  
    printf("%s ", column[i]);  
}  
printf("\\n");
```

# 5）结果集中行操作（记录操作）

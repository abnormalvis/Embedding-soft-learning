# 2. 网络调试工具使用

# 2.1 软件介绍

Wireshark（前称Ethereal）是一个非常好的网络封包分析软件工具。网络包分析工具的主要作用是尝试捕获网络包，并尝试显示包的尽可能详细的情况。Wireshark使用WinPCAP作为接口，直接与网卡进行数据报文交换。

# 2.2 简单使用

2.1 打开 wireshark 界面。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-03-20/972edad8-2f03-4bde-a593-d3f177302fb8/6253a88611ca1e551436b1e53888e17f8df3651c6da5f128228555dcb124049e.jpg)


2.2 打开 capture——>option, 配置捕获滤波器。配置界面有五个部分:

Capture:在这个部分你可以配置你所抓取得网络接口。

capture File: 可以配置文件的保存路径，还可以通过 Use multipart File 选项实现自动广州小蚁智控科技有限公司 ● 创训工场

分批保存。

Stop capture：设定结束抓取的标志，如设置1000个packets后停止抓取。

Display Option: 设置报文显示方式。

Name Retouution:MAC 地址重定向。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-03-20/972edad8-2f03-4bde-a593-d3f177302fb8/07249d29a0e919036cc7139b7940c8118570c3d7e1566d5db686a20e09022083.jpg)
